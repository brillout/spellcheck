# `@brillout/spellcheck`

## Ignore comments

Next line:

```js
// spellcheck-ignore
```

Block:

```js
// spellcheck-ignore:on
// spellcheck-ignore:off
```

Whole file:

```js
// spellcheck-ignore:file
```

Any comment delimiter works:

```js
/* spellcheck-ignore */
```
```yaml
# spellcheck-ignore */
```
```html
<!-- spellcheck-ignore -->
```
